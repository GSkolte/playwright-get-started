# Playwright Get Started

This repository contains a simple Playwright test project.

## Structure

```
playwright-get-started/
├─ package.json             # npm metadata & scripts
├─ playwright.config.ts     # configuration for `playwright test`
├─ tests/                   # primary test directory
│   └─ example.spec.ts      # example tests
├─ playwright-report/       # HTML reports generated by Playwright
├─ test-results/            # raw test outputs (traces, videos, etc.)
└─ README.md
```

You can run the suite with:

```bash
npm install
npm run test
```

Reports are written to `playwright-report/` by default (`npm run test:report`).
